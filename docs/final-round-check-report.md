### 部署目录
```

├── blog-backend-1.0.0.jar
├── uploads/          # 文件上传目录
├── logs/             # 日志目录
└── backup/           # 备份目录
```

### 环境变量
```bash
DB_HOST=localhost
DB_PORT=5432
DB_NAME=nanmuli_blog
DB_USER=blog_user
DB_PASSWORD=your_password
REDIS_HOST=localhost
REDIS_PORT=6379
```

### 部署步骤
1. 创建数据库并执行 `init.sql`
2. 配置环境变量
3. 启动后端: `java -jar blog-backend-1.0.0.jar --spring.profiles.active=prod`
4. 部署前端静态文件到Nginx
5. 配置Nginx反向代理到8081端口
