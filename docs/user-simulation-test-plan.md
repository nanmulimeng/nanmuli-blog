# 项目模拟真实用户使用测试计划

> 第七轮测试：真实用户场景模拟 | 2026-04-06

## 一、测试目标

模拟真实用户从访问到使用的完整旅程，验证系统在实际使用场景下的表现。

## 二、用户角色定义

### 角色1：访客 (Visitor)
- 未登录状态
- 只能访问公开接口
- 行为：浏览文章、查看分类、搜索内容

### 角色2：管理员 (Admin)
- 已登录状态
- 可访问管理接口
- 行为：内容管理、系统配置

## 三、测试模块与场景

### 模块A：访客浏览流程测试
**测试代理**：`user-test-visitor`

场景清单：
1. 首页访问 → 文章列表 → 文章详情 → 返回列表
2. 分类筛选 → 切换分类 → 查看分类下文章
3. 归档页面 → 按年月浏览历史文章
4. 搜索功能 → 输入关键词 → 查看结果
5. 技术日志 → 时间线浏览 → 日志详情
6. 项目展示 → 技能云 → 关于页面
7. 快速切换页面测试（验证请求取消机制）

验证点：
- 页面加载速度 (< 3s)
- 数据渲染正确性
- 主题切换无闪烁
- 返回/前进按钮行为正常
- 重复请求是否正确取消

---

### 模块B：管理员工作流测试
**测试代理**：`user-test-admin`

场景清单：
1. 登录流程
   - 正确凭证登录 → 跳转管理后台
   - 错误密码 → 错误提示
   - Token过期 → 自动跳转登录页

2. 文章管理完整流程
   - 创建文章 → 填写内容 → 发布 → 查看前台
   - 编辑文章 → 修改内容 → 保存 → 验证更新
   - 置顶文章 → 验证前台置顶显示
   - 删除文章 → 验证前台不可见

3. 分类管理
   - 创建父分类 → 创建子分类
   - 验证非叶子分类不能关联文章

4. 技术日志管理
   - 创建日志 → 验证前台时间线
   - 编辑日志 → 验证更新

5. 项目/技能管理
   - 新增项目 → 前台展示验证
   - 新增技能 → 技能云更新验证

验证点：
- 权限控制（未登录访问管理页）
- 表单验证提示
- 操作成功/失败反馈
- 数据一致性（前后台同步）

---

### 模块C：边界场景与异常处理测试
**测试代理**：`user-test-edge`

场景清单：
1. 网络异常
   - 请求中断 → 错误提示
   - 后端服务不可用 → 友好错误页面
   - 慢网络 → 加载状态显示

2. 数据边界
   - 超长标题文章（200字符边界）
   - 超大内容文章（字数统计验证）
   - 空分类（无文章时的展示）
   - 大量分类（树形结构渲染）

3. 并发操作
   - 多标签页登录状态同步
   - 快速重复提交（防抖验证）

4. 特殊输入
   - XSS尝试（<script>alert(1)</script>）
   - SQL注入尝试（' OR '1'='1）
   - 特殊字符标题

验证点：
- 安全防护有效性
- 错误提示友好性
- 系统稳定性

---

### 模块D：跨模块数据一致性测试
**测试代理**：`user-test-integration`

场景清单：
1. 文章-分类关联
   - 删除有文章的分类 → 系统行为
   - 修改分类 → 关联文章同步

2. 文章-标签关联
   - 文章标签增删 → 标签云更新

3. 仪表盘统计
   - 发布文章 → 仪表盘数据更新
   - 删除文章 → 统计同步

4. 文件上传
   - 上传封面图 → 文章关联 → 删除文章 → 文件处理

验证点：
- 关联数据一致性
- 统计数据准确性
- 缓存失效正确性

---

## 四、测试执行步骤

每个代理执行以下步骤：

1. **代码审查**：检查相关模块代码
2. **场景模拟**：按照场景清单验证
3. **问题记录**：发现问题记录到报告
4. **修复建议**：提出修复方案

|-----|------|---------|---------|
| 1 | xxx | 高/中/低 | xxx |

## 六、执行顺序

1. 并行执行模块A、B、C、D
2. 汇总问题统一修复
3. 回归验证
